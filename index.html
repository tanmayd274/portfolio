About Me

My journey into cybersecurity began not in a server room, but in a code editor. As a software engineer with 1.5 years of experience, I was fascinated by how applications were built using Python, JavaScript, and cloud technologies.

I became obsessed with how those same applications could be broken—and, more importantly, how to fortify them. This passion led me to pursue graduate-level training in cybersecurity, earning my CompTIA Security+ and ITIL Foundation certifications. I now specialize in vulnerability assessment, penetration testing, and securing the entire development lifecycle.
