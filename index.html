About Me

My journey into cybersecurity began not in a server room, but in a code editor. As a software engineer, I was fascinated by how applications were built, but I became obsessed with how they could be broken—and, more importantly, how to fortify them. This passion led me to pursue a Master's degree and specialize in the area where development and security converge: the cloud.

I thrive on the command line and in the cloud console, using tools like Docker, Kubernetes, and Jenkins to build streamlined, secure development pipelines. My hands-on experience at Infopercept Consulting involved engineering compliance platforms for major financial institutions, where I didn't just write code; I built secure, scalable solutions on AWS and Azure.
